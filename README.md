# Rainwave Streamer

This is a temporary repository to hold Rainwave's new audio streamer until it can be merged into the main repository.

Debian requirements:

```sh
apt install ffmpeg libshout3-dev libgirepository-2.0-dev libcairo2-dev pkg-config python3-dev gir1.2-gstreamer-1.0 gstreamer1.0-tools gstreamer1.0-plugins-base gstreamer1.0-plugins-good gstreamer1.0-libav
```
